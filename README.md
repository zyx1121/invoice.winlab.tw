# Invoice

Invoice management system for NYCU WinLab — upload, parse, and organize invoices.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Backend**: Supabase
- **Styling**: Tailwind CSS + shadcn/ui
- **PDF**: pdfjs-dist + jszip
- **Package Manager**: Bun

## Getting Started

```bash
bun install
bun dev
```
